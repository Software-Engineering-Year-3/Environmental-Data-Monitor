ViewData Documentation

This documentation provides an overview of the code for the first issue: "View and analysise historical data".

Overview

This issue focuses on allowing the appropriate user to view and analysise the data given by the sensors, in order to study and recognise patterns, trends and potential future risks.

Key Components

  • ViewData.xaml.cs:It displays air quality, water quality, and weather quality data using charts. The data is fetched from a mock database and rendered dynamically.
  • ViewData.xaml: It defines the UI layout of the dashboard - it uses a "ScrollView" and "VerticalStack" to ensure an organised sections for air quality, water, etc..

XAML Structure


	lt;ScrollView>
		<VerticalStackLayout Padding="20">
			<!-- Air Quality Section -->
			<Label Text="Air Quality Data" FontSize="20" FontAttributes="Bold" HorizontalOptions="Center" />
			<StackLayout x:Name="AirQualityView" HeightRequest="300" BackgroundColor="LightGrey" />

			<!-- Water Quality Section -->
			<Label Text="Water Quality Data" FontSize="20" FontAttributes="Bold" HorizontalOptions="Center" />
			<StackLayout x:Name="WaterQualityView" HeightRequest="300" BackgroundColor="LightGrey" />

			<!-- Weather Quality Section -->
			<Label Text="Weather Quality Data" FontSize="20" FontAttributes="Bold" HorizontalOptions="Center" />
			<StackLayout x:Name="WeatherQualityView" HeightRequest="300" BackgroundColor="LightGrey" />
		</VerticalStackLayout>
	</ScrollView>

Key Elements

  • ScrollView:Ensures the page can be scrollable if the content occupies more that the screen height space
  • VerticalStackLayout:Stacks sections of the dashboard vertically
  • StackLayout: It is used in each data section in order to dynamically stack charts

XAML.cs Methods

  • LoadDashbardData:Fetches data for all the data types and calls the respective chart generation methods
  • GenerateLineChart:The helper reusable method to create a "GraphicsView" with a line chart for all the data types
  • GenerateAirQualityChart:Generates charts for air quality data
  • GenerateWaterQualityChart:Generates charts for water quality data
  • GenerateWeatherQualityChart:Generates charts for weather quality data
